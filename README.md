Simple puzzle game

install the repo with npm
npm install

start the frontend and json-server
npm run full

start the frontend
npm run start

start the json-server
npm run server

start watching test
npm test

go to [http://0.0.0.0:9000](http://0.0.0.0:9000) or [http://localhost:9000](http://localhost:9000) in your browser
